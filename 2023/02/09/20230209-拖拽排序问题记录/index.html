拖拽问题

在 vue2 项目中使用 sortable.js 拖拽时，每次拖拽的顺序都比较混乱。

导致原因:
直接操作真实 dom 时，与虚拟 dom 不匹配。
虽然绑定了 key 值，但是渲染的时候

简单说就是改变真实 dom 位置时，虚拟 dom 没有变化，虚拟 dom 重新渲染的时候还是上一次的顺序。

解决：
在拖拽 end 事件内

或者使用 v-if 重新渲染节点
